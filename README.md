# m-wellbet-hub

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容彼此独立，不针对任何特定域名或网站。

## 项目简介

这是一个用于存放和展示 HTML 静态页面的集合仓库。每个页面可以单独访问，主要用于资料归档和展示用途。仓库本身不提供动态服务，也不包含后端逻辑。

## 目录说明

- `index.html`：仓库主页，展示页面列表或导航。
- `pages/`：存放各个独立 HTML 页面的子目录。每个页面建议以有意义的文件名命名，便于识别。
- `assets/`：存放页面所需的公共资源，如样式表、脚本、图片等（可选）。

## 页面归档说明

- 每个 HTML 页面均为独立完整的文件，可脱离本仓库在任意静态服务器上直接运行。
- 页面之间不存在相互依赖关系，除非明确引用公共资源。
- 归档的页面内容可能涉及多种主题，仅作为资料留存，不构成对任何外部内容的背书。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面。
- 欢迎提交 Issue 或 Pull Request 进行内容补充或修正，但需确保所提交内容不违反相关法律法规及平台政策。
- 维护者保留对仓库内容的最终解释权和修改权。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/m-wellbet-hub.git
   ```
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件即可查看。
3. 也可部署到任何静态托管服务（如 GitHub Pages）进行在线访问。
